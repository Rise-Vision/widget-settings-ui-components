Widget Button Toolbar

An Angular directive that provides the set of buttons required for a widget settings file.

Dependencies N/A

Usage

The directive provides attributes to set the value of the href for the Help button and Contribute buttons. Further attributes are also provided to allow setting the external function to run for the click event of the Save and Cancel buttons. Lastly another attribute allows for executing external code to return a boolean value that determines the disabled state of the Save button.
