Background Image Setting

An Angular directive that allows a user to format the background of a widget with a color and/or an image

Dependencies angular-bootstrap-colorpicker UI Bootstrap Tooltip Storage Selector URL Field Position Setting

Usage

The directive provides controls for choosing an RGBA value to use as a Background color as well as an specifying an image (optional).

When opting to use an image, you can either enter a valid URL to an existing image on a server or use the Storage Selector and choose a file you have stored in your Rise Vision Storage. If you intend on choosing from your Storage, be sure to include the company-id attribute and your company id as the value like so company-id="xxxxxxx". Please review the usage of the Storage Selector for more information regarding company-id.
